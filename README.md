## Controls - Mouse

Can be played with mouse only.

| Button      | Description          |
| ----------- | -------------------- |
| Left Click  |  Build / Drag Items  |
| Right Click | Destroy              |
| Escape      | Menu                 |
| Numbers 1-7 | Select Tool Shortcuts|
